## Description

Ce projet est un minuteur simple qui permet à l'utilisateur de saisir un nombre de minutes et de lancer un compte à rebours correspondant.

## Utilisation

1. Ouvrez le fichier `index.html` dans un navigateur web moderne.

2. Saisissez le nombre de minutes dans le champ prévu à cet effet.

3. Cliquez sur le bouton "GO" pour démarrer le minuteur.

4. Le minuteur affichera le temps restant sous la forme de minutes et de secondes décroissantes.

5. Une fois que le temps est écoulé, le minuteur affichera "C'est terminé !" et s'arrêtera automatiquement.
